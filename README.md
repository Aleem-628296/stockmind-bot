# StockMind — Inventory Management Telegram Bot

A tap-to-use inventory management system built for **Victory Venture**, a phone accessories shop in Accra, Ghana.
🌐 **Live Demo:** [stockmind-bot.onrender.com](https://stockmind-bot.onrender.com)

## What It Does
- Browse stock by category (iPhone Screens, Samsung Screens, Chargers, Accessories)
- Record sales with quantity and customer tracking
- Credit sales management with automatic payment reminders every 2 hours
- Add new stock with item name, color, quantity, cost price, selling price, and category
- Multi-user support: Owner (full access) + Secretary (limited access)
- Daily summary with sales count, profit, and pending payments

## Tech Stack
- **Backend:** Python, Flask
- **Messaging:** Telegram Bot API
- **Database:** SQLite
- **Deployment:** Render (free tier)

## Impact
- Replaced error-prone manual ledger system
- Improved cash flow visibility with credit tracking
- Secretary trained and actively using the system

## Status
✅ Live and deployed. In active use by the business owner and staff.
